# Benazir Educational Centre

Benazir Educational Centre is a modern, responsive educational web platform for learning English, Russian, Chinese and web development.

## Features

- Responsive design for mobile, tablet and desktop
- Light and dark mode
- Tajik and Russian language switcher
- Course catalog with filters
- Student dashboard
- AI assistant with prepared educational answers
- Contact page with map and feedback form
- Minimalist logo and glassmorphism UI

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
- GitHub Pages

## Pages

- Home page
- Courses catalog
- Student dashboard
- About and contacts

## Project Goal

The goal of this project is to digitalize the learning experience for educational centers and make course registration, student support and information access easier.
